# chromeos双拼输入法

根据[google-input-tools](https://github.com/google/google-input-tools/tree/master/chrome/os/ime)改造，

## 功能
-[x] 整句输入
-[x] 模糊音
-[x] 用户词典

## 使用
chrome打开`开发人员模式`，
从[release]()下载编译好的扩展，解压后，在扩展页面`加载解压缩的扩展`，
在输入法设置中添加`双拼输入法`。
## 编译
```shell
# 安装依赖
pnpm i

# 编译，生成扩展在dist文件夹中
pnpm run build
pnpm run build-background
pnpm run build-option
```
